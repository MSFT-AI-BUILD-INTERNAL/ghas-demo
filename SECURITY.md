# 보안 정책 (Security Policy)

이 저장소는 GitHub Advanced Security(GHAS) 기능을 시연하기 위한 **데모**입니다.
`app/` 코드에는 교육 목적으로 취약점이 **의도적으로** 심어져 있습니다. 실제 운영에 사용하지 마세요.

## 지원 범위

| 대상 | 지원 |
|---|---|
| `main` 브랜치 | ✅ |
| 그 외 | ❌ (데모) |

## 취약점 신고

실제 취약점을 발견하면 이슈로 공개하지 말고
[Security Advisory](../../security/advisories/new) 로 비공개 신고해 주세요.

## 이 저장소가 시연하는 보안 통제

- Code Scanning (CodeQL + 커스텀 쿼리) · Copilot Autofix
- Dependabot alerts / security updates / Dependency Review
- Secret Scanning · Push Protection · AI(generic) 탐지
- Rulesets (스캔 통과·리뷰 필수) · Security Overview · Campaigns
